# Numato 32-Channel Ethernet Relay Board Control Demo

## Overview
This project demonstrates the working and control process of the Numato 32-Channel Ethernet Relay Board using TCP/IP communication.

The demo includes:
- Ethernet relay board overview
- Authentication before rebooting
- Device reboot process
- Relay control after reboot
- IP verification using Command Prompt
- Relay control through browser using TCP/IP protocol
- Web-based relay switching using Numato interface

## Features
- 32-channel relay control
- Ethernet-based communication
- TCP/IP protocol support
- Secure authentication before reboot
- Remote relay switching
- Browser-based control interface
- Network connectivity testing using ping command

## Hardware Used
- Numato 32-Channel Ethernet Relay Board
- Ethernet cable
- Windows PC/Laptop
- Network connection

## Software Used
- Command Prompt (CMD)
- Web Browser
- TCP/IP Communication
- Numato Official Web Interface

## Working Process
1. Connect the Numato relay board to the network.
2. Configure the IP address.
3. Authenticate the user before rebooting.
4. Reboot the device after successful authentication.
5. Verify connectivity using the ping command.
6. Open the board IP address in the browser.
7. Control relays using the web interface.
